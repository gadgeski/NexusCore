# NexusCore ⚡

**Brain Buffer for Android.**
脳のメモリを解放する、無骨な「掃き溜め」ツール。

## 🇯🇵 概要 (Concept)

### 1. アプリの定義

**「サイバーパンク・コンテキスト・スイッチング・リポジトリ」**

(Cyberpunk Context-Switching Repository)

### 競合との違い

- **VS Obsidian / Notion:**
  - これらは「整理・構造化・リンク」に特化した**「母艦」**です。
  - NexusCoreは、母艦に送る前の**「迎撃機」**です。整理不要、タグ付け不要（モードがタグ代わり）、最速で思考をキャプチャし、母艦へ転送するための一次バッファとして機能します。
- **VS Google Keep / 標準メモ:**
  - これらは「誰にでも使いやすい文房具」です。
  - NexusCoreは**「選ばれたエンジニアのための尖った武器」**です。警告色のUI、グリッチノイズ、黒背景。これらに美学を感じるユーザーだけが、このアプリの真価を引き出せます。

思考の断片、URL、突然の思いつき……整理されていない情報を、Android の「共有」メニューから即座に投げ込んでください。整理は後でやればいいのです。

- **Cyberpunk / Neo-Brutalism UI:** 丸みのないデザイン、高コントラスト、そして動的なデジタルノイズ。
- **Share-First:** アプリを開いて書くのではなく、他アプリからの「共有」で受け取ることに特化。
- **Clean Architecture:** 見た目はパンクでも、中身は堅牢なモダン Android 設計。
